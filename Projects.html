System Analysis and Testing

The first project I worked on was a group project on for my 'System Analysis and Testing' module.
This was a two part project, with the first phase requiring an individual hand up containing analysis of several software systems of a specific narture. I chose food ordering and delivery services, and so analysed all of their available functionality. This included creating Use Case Diagrams for all of the available actions, Activity Diagrams to lay out the steps of each function, and Entity Classes for system operators.

The second phase for this project was a group element. We were required to work as a team to combine our individual projects, improve upon individual aspects as well as create Test Cases and Sequence Diagrams for more functions of the site.

Information Systems

For my 'Infromation Systems' project I was required to assess a Customer Relationship Management company and investigate Database Management Systems.

I was required to detail different types of database management systems which were availableto the company and suggest onw which was best suited to their company description. I was also required to design an Entity Relationship Diagram to demonstrate how a database for this company would look.

OOSD

I am in the process of creating a program for my 'Object Oriented Software Design' module.
The language we are using for creating our program is Python. Python is a programming language which is considered to be for beginners because it helps you with error correction and other useful features.

The program we have to design is to calculate the cheapest route between 5 airports factoring in the distance between the locations, the cost of fuel, and the currency exchange rate.
This project requires the use of object oriented principles and task management skills we covered throughout the year.
